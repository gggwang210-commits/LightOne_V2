# LIGHT ONE Healthcare Premium UI 분석 및 반영 내역

## 1. 기준 이미지에서 추출한 스타일

- 전체 톤: 고급 헬스케어 SaaS 대시보드
- 배경: 순백색이 아니라 아주 옅은 블루 그레이 계열의 의료 서비스형 배경
- 사이드바: 좌측 고정형, 브랜드 로고와 기능 메뉴를 명확히 분리
- 히어로: 짙은 네이비 그라데이션, 데이터 네트워크/의료 신호 느낌의 시각 효과
- 카드: 큰 radius, 얇은 border, 은은한 shadow, 색상은 과하게 쓰지 않음
- 폰트: Inter + Noto Sans KR 조합, 제목은 800~900 굵기, 본문은 500~700
- UX: KPI → 분석 → 리포트 → 우선순위 → 로드맵 순서로 읽히는 업무형 흐름

## 2. Netlify 레퍼런스 반영 방향

직접 URL fetch는 환경상 실패했으므로, 사용자가 제공한 캡처와 기존 레퍼런스 이미지를 기준으로 다음을 반영했습니다.

- 화면 상단의 넓은 시각 영역 강화
- 의료/AI 느낌의 네트워크 패턴과 신호형 배경 추가
- 카드 hover transition과 순차 등장 애니메이션 추가
- 기존 프로젝트 가이드형 화면에서 실제 SaaS 대시보드형 화면으로 전환
- BodyScan AI 중심 문구를 PT 상담 리포트 SaaS, QS/JATC, Human-in-the-loop 구조로 정리

## 4. 사업 메시지 반영

- AI 체형분석 단독 상품이 아니라 비의료 PT 상담 리포트 SaaS로 표현
- QS/JATC는 의료 판단 지표가 아니라 상담 참고 지표로 명시
- AUTO/REVIEW/BLOCK 라우팅은 AI 최종 판단이 아니라 트레이너 검토 우선순위로 표현
- 촬영 QC, 카메라 보정, 조명 정규화는 리포트 신뢰도 보조 레이어로 배치
