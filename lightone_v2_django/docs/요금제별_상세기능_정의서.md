## Starter 요금제 — ₩29,000 / 월

**타겟:** 혼자 운영하는 1인 PT 샵 원장 또는 프리랜서 트레이너.
**핵심 가치:** "지금까지 머릿속에만 있던 회원 기록을 처음으로 체계화한다."

### 계정 및 권한
- 트레이너 계정 1개 (본인만 사용)
- 회원 계정 최대 30명 등록
- 이메일/비밀번호 로그인

### 데이터 입력 기능
- 세션 기록 입력: 운동 종목, 세트/횟수/중량, RPE(운동자각도), NRS(통증 수치), 메모
- 회원 기본 정보 등록: 이름, 나이, 성별, 특이사항(부상 이력 등)
- 모바일 웹 브라우저에서 입력 가능 (별도 앱 불필요)

### QS 산출 및 위험 라우팅
- 세션 입력 후 자동 QS(품질 점수) 산출 (0~100점, 규칙 기반)
- 위험 라우팅 자동 분류: **AUTO** (일반 진행) / **REVIEW** (트레이너 검토 필요) / **BLOCK** (안전 신호, 전문가 상담 권고)
- REVIEW/BLOCK 발생 시 화면 내 알림 표시

### 대시보드 및 시각화
- 회원별 최근 5회 세션 QS 변화 추이 그래프
- 통증(NRS) 변화 추이 그래프
- 전체 회원 위험 라우팅 현황 요약 (AUTO/REVIEW/BLOCK 건수)

### 리포트 생성
- 월 50건 비의료 상담 리포트 생성
- 리포트 포함 항목: 기간 내 세션 요약, QS 변화, 통증 반응, 트레이너 메모
- 웹 화면 출력 (인쇄 가능)
- 모든 리포트 하단에 "본 리포트는 트레이너의 상담을 보조하기 위한 참고 자료이며, 의학적 진단을 대체하지 않습니다." 자동 삽입

### 고객 지원
- 이메일 지원 (영업일 기준 48시간 이내 응답)
- 온라인 사용 가이드 문서 제공

### 제한 사항 (Starter에서 불가)
- 트레이너 다중 계정 (Pro 이상)
- PDF 리포트 다운로드 (Pro 이상)
- 커스텀 로고 삽입 (Pro 이상)
- 센터 통합 대시보드 (Enterprise)

## Pro 요금제 — ₩89,000 / 월

**타겟:** 트레이너 3~5명이 함께 일하는 중소형 PT 샵 원장.
**핵심 가치:** "트레이너가 달라져도 PT 품질이 흔들리지 않는다. 회원 재등록률이 올라간다."

### Starter 전체 기능 포함 +

### 계정 및 권한
- 트레이너 계정 최대 5명 (원장 계정 포함)
- 원장 계정: 소속 트레이너 전체 데이터 열람 가능
- 트레이너 계정: 본인 담당 회원 데이터만 열람 가능 (권한 분리)
- 회원 계정 최대 150명 등록

### 데이터 입력 기능 (추가)
- 자세 관찰 체크리스트 입력 (어깨 높이 차이, 골반 기울기 등 항목 선택형)
- 촬영 품질 평가 입력 (조명, 각도, 화질 등 3단계 선택)
- 세션 목표 설정 및 달성 여부 기록

### QS 산출 및 위험 라우팅 (강화)
- Starter 기능 포함
- REVIEW 상태 세션에 대한 트레이너 승인(Approve)/메모 기능
- BLOCK 발생 시 원장 계정에 자동 알림 이메일 발송

### 대시보드 및 시각화 (강화)
- 회원별 전체 기간 QS 변화 추이 (기간 필터 적용 가능)
- 트레이너별 담당 회원 평균 QS 현황 비교 (원장 전용)
- 위험 라우팅 발생 빈도 히트맵 (요일/시간대별)
- 회원 재등록 예정일 알림 (수동 입력 기반)

### 리포트 생성 (강화)
- 월 300건 비의료 상담 리포트 생성
- **PDF 다운로드** 기능 (회원에게 파일로 전달 가능)
- **센터 로고 삽입** 기능 (브랜딩 강화)
- 리포트 유형 2가지:
  - **세션 리포트:** 단일 세션 결과 요약 (상담 직후 제공용)
  - **월간 리포트:** 한 달 전체 변화 요약 (재등록 상담 시 활용)
- 리포트 공유 링크 생성 (회원에게 URL로 전달 가능)

### 고객 지원 (강화)
- 이메일 지원 (영업일 기준 24시간 이내 응답)
- 온라인 사용 가이드 문서 제공
- 월 1회 30분 온라인 사용법 컨설팅 (화상 미팅)

## Enterprise 요금제 — ₩199,000 / 월

**타겟:** 다수 트레이너를 보유한 대형 피트니스 센터, 또는 여러 지점을 운영하는 PT 프랜차이즈 본사.
**핵심 가치:** "전 지점의 PT 품질을 본사에서 실시간으로 모니터링하고 표준화한다."

### Pro 전체 기능 포함 +

### 계정 및 권한
- 트레이너 계정 무제한
- 회원 계정 무제한
- **지점(Branch) 개념 추가:** 지점별 데이터 분리 및 본사 통합 열람
- 권한 레벨 3단계: 본사 관리자 > 지점 원장 > 트레이너

### 대시보드 및 시각화 (최상위)
- **지점 통합 대시보드:** 전 지점 QS 평균, 위험 라우팅 현황, 리포트 발행 건수 한눈에 비교
- **트레이너 KPI 대시보드:** 트레이너별 담당 회원 수, 평균 QS, REVIEW/BLOCK 발생률, 리포트 발행률 (성과 관리 지표)
- 데이터 기간 필터: 주간/월간/분기별 비교 가능

### 리포트 생성 (최상위)
- 리포트 생성 무제한
- **화이트라벨(White-label) 리포트:** LIGHT ONE 브랜드 완전 제거, 센터 자체 브랜드로 출력
- 리포트 템플릿 커스터마이징 (항목 추가/제거 가능)
- 분기별 회원 변화 종합 리포트 자동 생성

### 데이터 관리 (추가)
- 데이터 내보내기(Export): CSV/Excel 형식으로 전체 데이터 다운로드 가능
- 데이터 보관 기간: 무제한 (Starter/Pro는 2년)
- [로드맵] API 연동: 기존 센터 관리 시스템(ERP)과 데이터 연동

### 고객 지원 (최상위)
- 전담 계정 매니저 배정
- 이메일 + 전화 지원 (영업일 기준 4시간 이내 응답)
- 도입 초기 온보딩 지원 (현장 방문 또는 화상 교육)
- 월 2회 정기 리뷰 미팅

## 가격 책정 근거 및 방어 논리

### 경쟁 서비스 대비 포지셔닝
- 단순 PT 예약/기록 앱(예: 짐노트, 피트니스 관리 앱): 월 0~15,000원 수준, QS 산출 및 위험 라우팅 기능 없음.
- LIGHT ONE은 **"기록 도구"가 아닌 "의사결정지원 도구"**로 포지셔닝하여 가격 프리미엄 정당화.

### 고객 ROI (투자 대비 효과) 계산 예시
- **Pro 요금제 기준 (월 89,000원):**
  - 회원 1명의 PT 패키지 평균 금액: 약 600,000원 (10회 기준, [확인필요])
  - LIGHT ONE 도입으로 재등록률 10%p 향상 시: 150명 중 15명 추가 재등록 = 월 +9,000,000원 매출 효과 (추정)
  - 월 89,000원 투자로 최대 9,000,000원 효과 → ROI 100배 이상 (추정치, 파일럿 검증 필요)

> ※ 위 ROI 수치는 파일럿 검증 전 추정치입니다. 실제 파일럿 결과를 기반으로 업데이트 예정. [확인필요]
