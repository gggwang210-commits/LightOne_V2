# LIGHTONE V2 1페이지 사업 요약

## 1. 한 줄 소개

LIGHTONE V2는 PT샵·헬스장·필라테스 센터가 회원의 운동 기록, 생활습관, 통증 반응, 상담 메모를 구조화해 재등록 상담에 활용하도록 돕는 **비의료 PT 상담 보고서 SaaS**입니다.

## 2. 고객

| 고객 | 역할 | 핵심 니즈 |
|------|------|-----------|
| PT샵 대표 | 구매자·운영 책임자 | 상담 품질 표준화, 재등록 상담 자료, 팀 운영 관리 |
| 헬스장 대표 | 구매자·관리자 | 여러 트레이너의 기록 방식 통일, 상담 프로세스 관리 |
| 필라테스 센터장 | 구매자·현장 책임자 | 회원 변화 설명 자료, 수업 기록 정리, 브랜드 경험 강화 |
| 트레이너 | 실제 사용자 | 상담 준비 시간 단축, 회원에게 설명할 구조화된 리포트 |

## 3. 해결하려는 문제

1. 트레이너는 상담 전 자료 정리에 시간을 쓰고 있습니다. 구체 시간은 `[확인필요]`입니다.
2. 재등록 상담 때 회원 변화를 설명할 구조화된 자료가 부족합니다.
3. 센터장은 트레이너별 상담 품질 표준화에 관심이 있습니다.
4. 월 구독 또는 리포트당 과금에 대한 지불의사는 아직 검증이 필요합니다.
5. AI 단독 판단보다 트레이너 검토 기반 보고서가 더 수용 가능하다는 가설을 검증해야 합니다.

## 4. 솔루션

- 운동 기록, RPE, 통증 반응, 생활습관, 트레이너 메모를 한 화면에서 정리합니다.
- 상담 보고서 초안을 생성해 재등록 상담에서 회원 변화 설명 자료로 사용합니다.
- 리포트는 트레이너가 검토한 뒤 확정합니다.
- 센터장은 표준 양식으로 트레이너별 상담 품질을 확인할 수 있습니다.
- 파일럿 단계에서는 동의 기반, 비식별화, 최소수집 원칙을 적용합니다.

## 5. 제품 범위와 비의료 경계

LIGHTONE V2는 운동 상담과 기록 정리를 돕는 비의료 웰니스·피트니스 SaaS입니다. 통증 관련 정보는 운동 중 반응 기록으로만 다루며, 최종 상담 문구는 트레이너가 검토합니다. 임상적 판단, 질병 예측, 의학적 효과 주장, 자동 결론 제시는 제품 범위에 포함하지 않습니다.

## 6. 비즈니스 모델 가설

| 모델 | 설명 | 상태 |
|------|------|------|
| 월 구독 | 센터 또는 트레이너 계정 기반 SaaS | 가격 `[확인필요]` |
| 리포트당 과금 | 상담 리포트 생성 건별 과금 | 가격 `[확인필요]` |
| 파일럿 패키지 | 제한 기간 운영과 피드백 회고 포함 | 가격 `[확인필요]` |
| 교육·운영 패키지 | 센터 상담 양식 도입과 트레이너 온보딩 | 가격 `[확인필요]` |

## 7. 검증 계획

- PT샵 대표, 헬스장 대표, 필라테스 센터장, 트레이너를 대상으로 고객 인터뷰를 진행합니다.
- 상담 전 자료 정리 시간, 재등록 상담 자료 부족, 상담 품질 표준화 니즈를 확인합니다.
- 월 구독과 리포트당 과금에 대한 지불의사를 검증합니다.
- 파일럿 센터는 동의 기반, 비식별화, 최소수집 원칙을 수용하는 곳만 선정합니다.
- 핵심 문서는 [트레이너·센터 고객 인터뷰 질문지](../validation/trainer-interview-script.md), [고객 발견 인터뷰 로그 템플릿](../validation/customer-discovery-log-template.md), [WTP 테스트 계획](../validation/wtp-test-plan.md), [파일럿 센터 리스트 템플릿](../validation/pilot-center-list-template.md)을 사용합니다.

## 8. 현재 상태와 다음 단계

| 항목 | 상태 | 다음 액션 |
|------|------|-----------|
| 제품 정의 | 비의료 PT 상담 보고서 SaaS로 정리 | 고객 언어로 재검증 |
| 고객 인터뷰 | 준비 단계 | 3~5명 이상 인터뷰 `[확인필요]` |
| WTP 검증 | 준비 단계 | 월 구독·리포트당 과금 반응 확인 |
| 파일럿 후보 | 준비 단계 | 센터 리스트업과 조건 확인 |
| 제출 자료 | 초안 단계 | 인터뷰 결과 반영 후 업데이트 |

## 9. 핵심 메시지

LIGHTONE V2는 AI가 결론을 대신 내리는 서비스가 아니라, PT 현장의 상담 기록과 재등록 상담을 구조화하는 비의료 보고서 SaaS입니다. 초기 목표는 실제 고객 문제, 지불의사, 파일럿 운영 가능성을 빠르게 검증하는 것입니다.
