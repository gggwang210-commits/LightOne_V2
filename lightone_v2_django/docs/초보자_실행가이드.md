# LIGHT ONE V2 초보자 실행 가이드 🚀

이 문서는 Windows에서 LIGHT ONE V2 Django 프로젝트를 처음 실행하는 분을 위한 최신 기준 안내입니다.

---

## 1. 준비물 설치

1. **Python 설치**
   - [Python 공식 홈페이지](https://www.python.org/downloads/)에서 다운로드합니다.
   - 설치 화면에서 **Add Python to PATH**를 반드시 체크한 뒤 설치합니다.
2. **VS Code 설치**
   - [VS Code 공식 홈페이지](https://code.visualstudio.com/)에서 설치합니다.

---

## 2. 프로젝트 열기

1. 프로젝트 압축 파일을 받았다면 압축을 풉니다.
2. VS Code를 실행합니다.
3. `파일(File)` > `폴더 열기(Open Folder)`를 클릭합니다.
4. 저장소 안의 `lightone_v2_django` 폴더를 엽니다.

> 구버전 안내: 예전 자료에 `LIGHTONE_V2_Complete.zip`, `lightone_django_complete.zip`, `lightone_django` 같은 이름이 나오더라도 현재 실행 기준 폴더는 `lightone_v2_django`입니다.

---

## 3. 터미널 열기

VS Code 상단 메뉴에서 `터미널(Terminal)` > `새 터미널(New Terminal)`을 클릭합니다.

터미널에서 현재 위치에 `manage.py`와 `requirements.txt`가 있는지 확인합니다.

```powershell
dir manage.py, requirements.txt
```

저장소 루트에서 터미널을 열었다면 먼저 아래 명령으로 Django 폴더로 이동합니다.

```powershell
cd lightone_v2_django
```

---

## 4. 한 줄씩 실행하기

### 1단계: 가상환경 만들기

```powershell
python -m venv .venv
```

### 2단계: 가상환경 켜기

PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```

보안 오류가 나면 VS Code 터미널 종류를 Command Prompt(CMD)로 바꾸고 아래 명령을 사용하세요.

```cmd
.venv\Scripts\activate.bat
```

성공하면 터미널 왼쪽에 `(.venv)` 표시가 나타납니다.

### 3단계: 필요한 패키지 설치하기

```powershell
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

### 4단계: 데이터베이스와 더미 데이터 준비하기

```powershell
python manage.py migrate
python manage.py seed_lightone
```

`seed_lightone`은 데모 데이터와 로그인 계정을 함께 만듭니다.

---

## 5. 서버 실행하고 접속하기 🎉

```powershell
python manage.py runserver
```

터미널에 `Starting development server at http://127.0.0.1:8000/` 문구가 보이면 성공입니다.

브라우저 주소창에 아래 주소를 입력하세요.

```text
http://127.0.0.1:8000/lightone/
```

---

## 자주 묻는 질문

### `python` 명령을 찾을 수 없다고 나와요.

Python 설치 시 **Add Python to PATH**를 체크하지 않았을 가능성이 큽니다. Python을 다시 설치하면서 해당 옵션을 체크하세요.

### 가상환경 켜기에서 보안 오류가 나요.

PowerShell 실행 정책 문제일 수 있습니다. VS Code 터미널 우측 상단의 화살표 메뉴에서 **Command Prompt**를 선택한 뒤 아래 명령을 실행하세요.

```cmd
.venv\Scripts\activate.bat
```

### 서버를 끄고 싶어요.

터미널에서 `Ctrl + C`를 누르면 서버가 종료됩니다.

### 로그인 비밀번호가 맞지 않아요.

아래 명령을 다시 실행하면 더미 계정 비밀번호가 최신 기준으로 재설정됩니다.

```powershell
python manage.py seed_lightone
```
