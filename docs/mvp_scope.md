# LIGHT ONE MVP 범위

## 1. MVP 목표

MVP의 목표는 “AI가 몸 상태를 판단한다”가 아니라, PT 현장의 상담 기록과 세션 데이터를 구조화하고 트레이너 검토용 비의료 상담 리포트를 생성하는 최소 제품을 만드는 것이다.

## 2. MVP 포함

| 기능 | 포함 기준 | 비고 |
|---|---|---|
| 회원 등록 | 회원 코드, 기본 상담 상태, 동의 상태를 등록 | 실제 개인정보는 최소 수집 |
| 운동 목표 입력 | 회원의 운동 목적, 상담 목표, 기대 결과를 기록 | 의료 목표가 아닌 운동·컨디셔닝 목표로 표현 |
| 통증 반응 입력 | 부위, 강도, 빈도, 운동 전후 변화 등 반응을 기록 | 통증 원인 결정 금지 |
| 세션 운동 기록 | 운동명, 세트, 반복, 중량, 수행 메모 기록 | 상담 자료 구조화 목적 |
| RPE 입력 | 세션별 자각운동강도 기록 | 트레이너 상담 참고 지표 |
| 트레이너 메모 | 회원 반응, 수행 관찰, 다음 상담 메모 기록 | 최종 검토 책임은 트레이너 |
| QS rule-based score | 규칙 기반 수행 품질 또는 상담 참고 점수 산출 | 임계값은 [검증필요] |
| AUTO/REVIEW/BLOCK routing | 일반, 검토 필요, 중단 권고 신호를 분류 | 의료 진단이 아닌 운영 라우팅 |
| PDF 또는 웹 상담 리포트 | 회원 상담에 활용할 비의료 리포트 생성 | 트레이너 검토 후 공유 |
| 선택적 촬영 QC 결과 | 각도, 조명, 거리, 전신 포함 여부 등 품질 상태 표시 | 가능하면 포함, 원본 미디어 저장 최소화 |

## 3. MVP 제외

| 제외 기능 | 제외 이유 | 안전 대체 |
|---|---|---|
| AI 자동 의료 진단 | 의료기기·진단 오해와 임상 검증 부담 | 비의료 상담 참고 리포트 |
| 질병 위험 예측 | 의학적 위험도 판정으로 오해될 수 있음 | 주의 신호와 트레이너 확인 필요 표시 |
| 자동 처방 | 운동 처방 자동 확정은 책임·안전 리스크가 큼 | 트레이너가 검토하는 운동 제안 초안 |
| 병원 진단 리포트 | 의료기관 제출용 진단 문서가 아님 | PT 상담용 웰니스 리포트 |
| 통증 원인 결정 | 통증 인과 판단은 의료 영역으로 오해될 수 있음 | 통증 반응 기록과 변화 메모 |

## 4. MVP 산출물

1. 회원·세션·통증 반응 입력 화면
2. RPE 및 트레이너 메모 기록 화면
3. QS rule-based score 계산 로직
4. AUTO/REVIEW/BLOCK 라우팅 규칙
5. PDF 또는 웹 상담 리포트 템플릿
6. 비의료 고지문과 금지 표현 검수표
7. 합성 또는 익명화 샘플 데이터
8. 개인정보 최소 수집·삭제 요청 처리 문서 초안
