QS/JATC 데모 참고값 안내

QS와 JATC는 트레이너가 회원 컨디션 변화와 세션 운영 흐름을 검토하기 위한 데모 참고값입니다. 의료 판단 기준, 진단 지표, 치료 필요성 판단 기준으로 사용하지 않습니다.

Session Workflow

세션 전

Today Sessions에서 예약 회원을 선택하고 최근 피드백, QS/JATC 참고값, REVIEW/BLOCK 라우팅 상태를 확인합니다.

세션 중

운동 난이도 조정, 동작 범위 조정, RPE, 특이 반응을 수행 기록으로 남기고 원인 판단 표현은 사용하지 않습니다.

세션 후

Report Drafts에서 초안을 검토하고 휴식·회복 안내와 컨디셔닝 과제를 안전한 표현으로 정리합니다.

주간 리뷰

주간 변화, 리포트 발송 상태, 다음 세션 계획, 재상담 포인트를 비교해 트레이너 노트에 저장합니다.

Trainer Routing

Review Queue

AUTO/REVIEW/BLOCK은 의료 위험 등급이 아니라 트레이너 검토 라우팅 상태입니다. 세션 운영 확인, 기록 보완, 안내 문구 검토를 돕기 위한 내부 워크플로우로만 사용합니다.

AUTO

기본 세션 흐름으로 진행 가능하되 트레이너가 최종 확인합니다.

REVIEW

통증 반응, 촬영 품질, 주관적 피로도 등 기록을 트레이너가 확인합니다.

BLOCK

진단이 아니며 세션 운영 보류와 전문가 상담 권고 안내가 필요한 caution signal로 취급합니다.

다음 세션 주의사항 저장
LIGHT ONE은 의료 진단·치료·처방을 제공하지 않습니다. 본 트레이너 대시보드는 합성 데이터 기반 비의료 웰니스 상담 지원 프로토타입이며, QS/JATC 및 AUTO/REVIEW/BLOCK 표시는 트레이너 검토와 세션 운영을 돕는 참고 정보입니다. 갑작스러운 심한 통증, 일상생활 제한, 기타 우려 증상이 있는 경우 전문기관 상담을 권합니다.
