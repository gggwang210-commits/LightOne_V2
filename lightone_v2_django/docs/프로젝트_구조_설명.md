# LIGHT ONE V2 프로젝트 구조 설명

이 문서는 LIGHT ONE V2 Django 프로젝트의 폴더 및 파일 구조를 설명합니다.

---

## 핵심 개념 설명

### QS (Quality Score, 품질 점수)란?
회원의 세션 데이터(통증 NRS, 운동자각도 RPE, 자세 점수)를 바탕으로 자동 계산되는 0~100점 점수입니다.
점수가 높을수록 세션 품질이 좋음을 의미합니다.

> 현재 QS/JATC 및 AUTO/REVIEW/BLOCK 임계값은 MVP 데모용 내부 초안이며 의료 판단 기준이 아닙니다. 파일럿 데이터와 전문가 검토 후 조정이 필요하고, BLOCK은 진단이 아니라 운동 세션 중단 및 전문가 상담 권고 신호입니다.

### 위험 라우팅 (AUTO / REVIEW / BLOCK)이란?
QS 점수와 통증 수치를 기반으로 다음 세션 진행 방향을 자동으로 분류합니다.
- **AUTO (초록):** 일반 진행 가능
- **REVIEW (노랑):** 트레이너 검토 후 진행 권장
- **BLOCK (빨강):** 안전상 중단 및 전문가 상담 권고 신호 (의료 진단 아님)

> 현재 QS/JATC 및 AUTO/REVIEW/BLOCK 임계값은 MVP 데모용 내부 초안이며 의료 판단 기준이 아닙니다. 파일럿 데이터와 전문가 검토 후 조정이 필요하고, BLOCK은 진단이 아니라 운동 세션 중단 및 전문가 상담 권고 신호입니다.

### 미들웨어(Middleware)란?
모든 페이지 접속 시 자동으로 실행되는 보안 검사입니다. 로그인하지 않은 사용자는 자동으로 로그인 페이지로 이동시킵니다.
