### 2.3 H_dashboard_detail_v1 — 식단·개인운동·생활코칭

**페이지 타이틀:** 식단 · 개인운동 · 생활코칭 리포트  
**리포트 유형:** 생활코칭 견본

**7개 섹션:**
1. 식단 요약 — 식단 기록(조금 더 필요), 단백질 섭취(잘하고 있어요), 수분 섭취(보통), 야식 관리(주의), 아침 식사(유지 추천)
2. 오늘 식단 가이드 — 아침(단백질 포함), 점심(탄수화물과 채소 균형), 저녁(과식 줄이기), 간식(견과류 또는 요거트)
3. 다음 개인운동 — 데드버그(코어), 힙브릿지(골반), 밴드 로우(어깨), 스쿼트(하체) 각 3세트, 통증 시 중단
4. 이번 주 생활코칭 — 1시간마다 일어나기, 물 1.5~2L, 수면 일정, 스트레칭, 통증 알리기
5. 트레이너 피드백 — 운동 습관 유지, 골반·코어 개선, 하체 안정성+수분 섭취 주의
6. 오늘 컨디션 & 피드백 남기기 — 컨디션 선택(좋음/보통/피곤함/아쉬워요) + 운동 후 느낌 + 식단 기록
7. 이번 주 컨디션 흐름 — 월~일 이모지 기반 컨디션 트래커

1. **촬영 조건 단순화** — 1주일마다 같은 시간대, 같은 공간·같은 조명, 일정한 조명 환경(자연광 or 고정 조명), 카메라 고정(거리·높이·각도 고정) → "변수 최소화로 정확도 향상"
2. **기준선에 양발 위치** — 양발을 기준선에 맞춤, 본인이 확인 자세로 기립 → "일관된 기준으로 오차 최소화"
3. **정면/측면 촬영** — 정면 상체/하체, 측면 상체/하체 → "전신 균형을 다각도로 기록"
4. **분석 항목 추출** — 어깨 기울기, 골반 기울기, 척추 정렬도, 무릎 정렬도, 발목 정렬도, 좌우 편차, 체형 변화지수 → "기울기·편차·변화·오차 기록"
5. **수치화 및 점수화** — 변화량 수치화(각 항목별 °, mm, %), 항목별 가중치 적용(종합도 반영), 종합 점수 산출(100점 만점) → "변화를 점수로 한눈에 파악"
6. **변화 추적 & 피드백** — 주차별 변화 그래프, 개선 가이드 제공 → "데이터 기반 맞춤 피드백"

**하단 보충 섹션:**
- 분석 항목 예시: 어깨 기울기, 골반 기울기, 척추 정렬도, 무릎 정렬도, 발목 정렬도, 좌우 편차 — 실제 사진 기반 시각 예시
- 점수화 예시 테이블: 항목별 측정값·정상범위·편차·가중치·점수 매트릭스 (어깨 기울기 2.1°/±2°/+0.1°/15%/92점 등)
- 변화 추적 대시보드 예시: 종합 점수 82/100, 주차별 변화 추이(1~4주차: 65→72→78→82), 전후 비교 사진, 트레이너 코멘트

## 4. 검증 결과

### 4.1 비의료 포지셔닝 검증 — PASS

모든 화면 하단에 면책 문구 확인됨: "운동 및 체형 관리 참고용이며 의료 진단 목적이 아닙니다." 의료 용어("진단", "치료", "질환", "처방") 사용 없음.

### 4.2 데이터 일관성 — PASS (1건 확인 필요)

10개 화면 모두 동일한 견본 데이터 세트(홍길동/송광일/82점/24.3%/27.8kg) 사용. `T_dashboard_exercise_v1`과 `T_dashboard_demo_v1`의 중복 여부 확인 필요.

### 4.3 UI 일관성 — PASS

네이비/블루 브랜드 컬러, 좌측 사이드바+메인 콘텐츠 레이아웃, 카드 컴포넌트 패턴이 전 화면에 걸쳐 일관 적용됨.

### 4.4 파일명 정리 — 완료

V1의 오타(`dashbord`, `dashborad`)가 V2에서 모두 정리됨. 버전 표기 `_v1`/`_v2` 통일됨.
