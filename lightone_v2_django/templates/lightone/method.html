QS/JATC 코드 방법론

QS/JATC 규칙 기반 산출 → AUTO/REVIEW/BLOCK 라우팅 → 트레이너 검토 → 상담 리포트 확정 흐름입니다.

현재 임계값은 MVP 데모용 내부 초안이며 PT 상담 참고 기준입니다. 파일럿 데이터와 전문가 검토 후 조정이 필요하고, REVIEW/BLOCK은 트레이너가 안전 신호를 확인한 뒤 진행 여부를 안내하는 흐름입니다.

def route_session(qs, pain, qc_status):
    if pain >= 7 or qc_status == "FAIL":
        return "BLOCK"
    if qs < 60 or pain >= 4 or qc_status == "CHECK":
        return "REVIEW"
    return "AUTO"
