# 파일럿 센터 리스트 템플릿

이 템플릿은 LIGHTONE V2의 고객검증과 파일럿 실행 후보 센터를 관리하기 위한 문서입니다. 파일럿은 동의 기반, 비식별화, 최소수집 원칙으로 운영합니다. 실행 기준은 [파일럿 검증 계획](pilot-validation-plan.md), 개인정보 기준은 [개인정보 보호 체크리스트](../governance/privacy-checklist.md), 지불의사 확인은 [WTP 테스트 계획](wtp-test-plan.md)을 참조합니다.

## 1. 운영 원칙

- 실제 회원 개인정보는 이 문서에 기록하지 않습니다.
- 센터 담당자 연락처는 승인된 별도 관리 도구에 저장합니다.
- 이 문서에는 센터 코드와 후보 상태만 기록합니다.
- 파일럿 참여 전 동의 범위, 수집 항목, 보관 기간, 삭제 요청 절차를 안내합니다.
- 수집 항목은 파일럿 목표 달성에 필요한 최소 범위로 제한합니다.
- 리포트는 트레이너 검토 후 상담 자료로만 사용합니다.

## 7. 파일럿 우선순위 결정

1. 문제 강도와 운영 가능성이 모두 높은 센터를 1순위로 둡니다.
2. 동의·비식별화·최소수집 원칙을 수용하지 않는 센터는 파일럿 대상에서 제외합니다.
3. 단순 무료 체험보다 주간 피드백과 지불의사 논의가 가능한 센터를 우선합니다.
4. 실제 회원 사례는 외부 제출 문서에 직접 넣지 않고, 비식별 요약과 가상 예시로만 변환합니다.
