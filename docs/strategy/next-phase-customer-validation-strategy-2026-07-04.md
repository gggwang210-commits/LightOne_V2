# LIGHTONE 다음 단계 전략서 — 고객검증에서 파일럿까지

작성 기준: 2026-07-04 KST  
대상 저장소: `gggwang210-commits/LightOne_V2`  
전략 범위: 고객검증 → 파일럿 후보 확보 → 제출자료 반영 → 1차 유료화 가설 검증

## 1. 핵심 결론

현재 LIGHTONE V2는 문서 정리 단계가 아니라 현장 검증 단계입니다. 고객검증 패키지 5개 문서는 준비되어 있고 정적 검사도 통과했습니다. 다음 2주 동안의 목표는 기능을 더 늘리는 것이 아니라 실제 트레이너와 센터장의 반응을 확보해 `[확인필요]` 항목을 증거로 바꾸는 것입니다.

우선순위:

1. 트레이너/센터장 3~5명 인터뷰
2. 샘플 리포트 1~3종 피드백 확보
3. 파일럿 가능 센터 1~3곳 후보화
4. 지불의사와 과금 방식 반응 확인
5. 모두의창업·멘토링 제출자료에 고객검증 증거 반영

## 2. 현재 위치

확보된 것:

- `LightOne_V2` 저장소 정리
- 고객검증·파일럿 실행 문서 5개
- 비의료 표현 기준
- 개인정보 최소수집 원칙
- 1페이지 사업 요약
- WTP 테스트 계획

부족한 것:

- 실제 고객 인터뷰 결과
- 지불의사 근거
- 파일럿 후보 센터
- 샘플 리포트 사용성 피드백
- 재등록 상담 활용 가능성 증거

## 3. 2주 전략 목표

- 인터뷰 후보 5명 리스트업
- 인터뷰 요청 메시지 발송
- 1차 인터뷰 최소 3명 완료
- 샘플 리포트 피드백 최소 3건 확보
- 파일럿 후보 센터 1곳 이상 발굴
- README §11의 고객 인터뷰 상태를 업데이트할 근거 확보

## 4. 고객검증 질문

### 4.1 문제 검증

- 상담 전 자료 정리에 실제로 시간이 드는가?
- 회원 변화 설명 자료가 부족한가?
- 재등록 상담에서 구조화된 리포트가 필요한가?

### 4.2 구매자 검증

- 실제 결제자는 대표, 센터장, 트레이너 중 누구인가?
- 트레이너 개인 결제 가능성이 있는가?
- 센터 단위 구독이 가능한가?

### 4.3 지불의사 검증

- 월 구독형이 나은가?
- 리포트당 과금이 나은가?
- 어떤 조건이면 비용을 낼 수 있는가?
- 가격은 검증 전까지 `[확인필요]`로 유지한다.

### 4.4 파일럿 검증

- 2주~4주 테스트가 가능한가?
- 동의 기반, 비식별화, 최소수집 원칙을 수용할 수 있는가?
- 트레이너가 리포트를 검토할 시간을 확보할 수 있는가?

## 5. 인터뷰 대상 전략

1차 인터뷰는 5명 후보 중 최소 3명 완료를 목표로 합니다.

권장 구성:

- PT샵 대표 1명
- 헬스장 운영자 또는 센터장 1명
- 필라테스 센터장 1명
- 소속 트레이너 1명
- 1인 프리랜서 트레이너 1명

우선순위는 결제 가능성이 있는 대표·센터장입니다. 실제 사용성은 트레이너가 판단하므로 대표와 트레이너를 함께 확인해야 합니다.

## 6. 샘플 리포트 활용 전략

인터뷰 전 샘플 리포트 1~3종을 준비합니다.

권장 샘플:

- 전체 대시보드형 회원 리포트
- 체형·컨디션 변화 리포트
- 식단·개인운동·생활코칭 리포트

확인할 반응:

- 회원 상담에 실제로 쓸 수 있는가?
- 너무 복잡하지 않은가?
- 회원이 이해하기 쉬운가?
- 재등록 상담에 도움이 되는가?
- 트레이너가 수정·검토할 부분은 무엇인가?

## 7. 파일럿 전환 기준

다음 조건 중 3개 이상이 충족되면 파일럿 후보로 분류합니다.

- 대표 또는 센터장이 직접 관심을 보임
- 샘플 리포트에 구체 피드백을 제공함
- 실제 회원 상담에 써볼 수 있다고 답함
- 2주~4주 테스트가 가능하다고 답함
- 동의 기반·비식별화·최소수집 원칙을 수용함
- 유료화 또는 후속 조건 논의가 가능함

## 8. 비의료 운영 기준

반드시 유지할 기준:

- 의료 진단, 치료, 재활 치료, 의학적 처방 표현 금지
- 질병 예측 또는 의학적 효과 주장 금지
- AI 단독 최종 판단 표현 금지
- `통증 원인 분석`이 아니라 `통증 반응 기록`으로 표현
- `AI가 판단`이 아니라 `트레이너 검토 보조`로 표현
- 인터뷰 기록은 비식별 코드로 관리
- 파일럿은 동의 기반, 비식별화, 최소수집 원칙으로 운영
