# Dashboard Sample User Flows

## Member flows

- Check today’s condition: open Home → read Today Summary → submit simple condition check.
- View weekly change: open Progress → review “이번 주 변화” → open report if needed.
- Read trainer feedback: open Home or Messages → read reviewed feedback → mark as confirmed.
- Submit pain response: open Condition → select body area and response level → save as non-medical record.
- Book next session: open Reservation → choose slot → confirm request.
- Pay membership/session fee: open Payment → review synthetic invoice → mark payment flow.
- View report history: open Report → filter by month → open trainer-reviewed report.
- Send request to trainer: open Messages → write request → wait for trainer reply.

## Trainer flows

- Prepare today’s session: Home → Today Sessions → Member Detail → previous feedback.
- Record session: Session Record → exercise log → RPE → pain response → notes.
- Review pain response: Member Detail → Pain Response & RPE → decide Good/Check/Review label.
- Create consultation report: Report Builder → review draft → edit safe copy → publish.
- Respond to member request: MemberRequestInbox → open request → answer with non-medical wording.
- Plan next solution: Next Session Plan → adjust difficulty/range/rest/conditioning task.
- Handle REVIEW/BLOCK: Review Queue → read caution signal → document trainer review → provide safe next action.
- Prepare re-registration consultation: Retention → candidate detail → summarize recent changes and goals.

## Admin flows

- Check center status: Overview → active members, sessions, reports, payments.
- Review trainer workload: Trainers → compare sessions and response queues → rebalance.
- Monitor payment issue: Payments → unpaid list → assign follow-up.
- Check re-registration pipeline: Re-registration Pipeline → filter expiring members → assign trainer.
- Review communication quality: Communications → response time and feedback → follow up.
- Manage role permission: Permissions → role matrix → change access and log action.
- Export non-sensitive report summary: Reports → aggregate export → remove identifiers → download summary.
