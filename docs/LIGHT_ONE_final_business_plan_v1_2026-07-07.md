# LIGHT ONE 최종 사업계획서 v1.0

## 비의료 PT 상담 리포트 SaaS + 데이터 기반 프리미엄 컨디셔닝 브랜드

작성일: 2026-07-07  
문서 목적: LIGHT ONE 사업아이템을 시장성, 심사 대응성, 기술 실행 가능성, 개인정보·규제 안전성 기준으로 정돈한 최종 사업계획서 초안  
문서 기준: 비의료 웰니스·운동상담 보조 서비스. 의료 진단·치료·처방·질병 예측 목적 아님.

---

## 0. 핵심 요약

LIGHT ONE의 최종 방향은 **“AI 체형·통증 분석 SaaS”가 아니라 “비의료 PT 상담 리포트 SaaS + 데이터 기반 프리미엄 컨디셔닝 브랜드 운영 시스템”**이다.

기존의 “AI가 체형과 통증을 분석한다”는 표현은 의료기기 오해, 정확도 검증 부담, 개인정보 리스크를 키운다. 반면 “PT 현장의 상담 기록, 통증 반응, 자세 변화, 운동 수행, 생활습관 데이터를 구조화해 트레이너 검토용 리포트를 만든다”는 방향은 고객 문제, 수익모델, 기술 구현, 규제 안전성을 동시에 설명할 수 있다.

최종 포지션은 다음으로 고정한다.

> LIGHT ONE은 PT샵·헬스장·필라테스 센터가 회원의 운동 수행, 통증 반응, 자세 변화, 생활습관 데이터를 구조화하고, 트레이너 검토 기반 상담 리포트를 생성하여 재등록 상담, 회원 유지, 프리미엄 컨디셔닝 프로그램 운영에 활용하도록 돕는 비의료 웰니스 SaaS이자 브랜드 운영 시스템이다.

## 2. 최종 제품 정의

### 2.1 제품명

- 외부 제품명: **LIGHT ONE Report**
- 시스템명: **LIGHT ONE Conditioning OS**
- 장기 브랜드명: **LIGHT ONE Premium Conditioning**

### 2.2 한 줄 정의

> PT 현장의 상담 데이터를 표준화하고, 촬영·운동·통증 반응 데이터를 기반으로 트레이너 검토용 비의료 상담 리포트를 자동 생성하는 Human-in-the-Loop 시스템

- 세션별 운동 수행 기록
- 통증 반응 기록
- 자세 변화 참고 지표
- 촬영 조건 QC
- QS/JATC 상담 참고 지표
- AUTO/REVIEW/BLOCK 라우팅
- 트레이너 검토용 상담 리포트
- 재등록 상담 자료
- 프리미엄 컨디셔닝 브랜드 운영 기반

## 3. 고객과 시장성

### 3.1 초기 고객 우선순위

| 우선순위 | 고객 | 구매 가능성 | 이유 |
|---:|---|---:|---|
| 1 | 프리미엄 PT샵 | 높음 | 상담 리포트와 재등록 설득에 직접 활용 가능 |
| 2 | 교정운동·컨디셔닝 센터 | 높음 | 통증 반응·자세 변화 기록 니즈가 큼 |
| 3 | 필라테스 센터 | 중간~높음 | 체형 변화 설명 자료 수요 존재 |
| 4 | 1인 트레이너 | 중간 | 니즈는 있으나 월 구독료에 민감 |
| 5 | 일반 헬스장 | 중간 이하 | 운영자가 비용 대비 효과를 엄격히 봄 |
| 6 | 병원·재활기관 | 장기 | 규제·임상·영업 장벽 높음 |
| 7 | 일반 소비자 앱 | 후순위 | 광고비, 리텐션, 신뢰 확보 난이도 높음 |

### 3.2 핵심 고객 문제

1. 회원의 운동 기록, 통증 반응, 사진, 상담 메모가 흩어져 있다.
2. 트레이너가 상담 전 자료를 정리하는 데 시간이 많이 든다.
3. 회원은 본인의 변화가 객관적으로 관리되고 있다고 느끼기 어렵다.
4. 원장은 트레이너별 상담 품질을 관리하기 어렵다.
5. 재등록 상담에서 “왜 계속 운동해야 하는지”를 설명할 구조화된 자료가 부족하다.

### 3.3 시장성 문장

> LIGHT ONE은 PT센터의 상담·기록·리포트·재등록 과정을 데이터화하여, 트레이너의 설명력을 높이고 센터의 회원 유지와 프리미엄 프로그램 전환을 돕는 비의료 웰니스 SaaS입니다.

## 4. 서비스 목표

### 4.1 최종 목표

> 데이터 기반 프리미엄 PT 브랜드를 운영할 수 있는 표준 시스템을 구축한다.

LIGHT ONE은 단순 SaaS 판매가 아니라, SaaS를 적용한 프리미엄 PT·컨디셔닝 브랜드 운영모델을 만드는 것을 장기 목표로 한다.

| 축 | 설명 |
|---|---|
| SaaS | 센터와 트레이너가 쓰는 상담 리포트·회원 변화 관리 시스템 |
| 브랜드 | LIGHT ONE 방법론을 적용한 프리미엄 PT·컨디셔닝 운영 모델 |

### 4.2 단계별 목표

| 단계 | 목표 | 결과물 |
|---|---|---|
| 0단계 | 문제 검증 | 트레이너·센터장 인터뷰, 샘플 리포트 피드백 |
| 1단계 | 리포트 MVP | 회원 입력폼, QS/JATC 초안, PDF/웹 리포트 |
| 2단계 | 파일럿 | 센터 1~3곳, 회원 10~30명 동의 기반 테스트 |
| 3단계 | 유료 PoC | 월 구독 또는 리포트 건당 과금 테스트 |
| 4단계 | 브랜드 패키지 | LIGHT ONE 컨디셔닝 프로그램, 교육·인증 |
| 5단계 | 확장 | SaaS 판매, 제휴형 도입, 기업 B2B 프로그램 |

## 6. 기술 전략

### 6.1 기술 포지션

기술의 핵심은 “딥러닝으로 통증을 맞힌다”가 아니다. 핵심은 다음 4가지다.

| 기술 | 목적 |
|---|---|
| Python/Django/FastAPI | MVP 웹서비스, API, 리포트 생성 |
| SQL DB | 회원·세션·지표·리포트 데이터 구조화 |
| Pose Estimation | 자세 keypoint 추출, 전후 변화 참고지표 |
| ML/DL | 향후 QS/JATC 보정, 패턴 분석, 리뷰 우선순위화 |

### 6.2 기술 차별화

> 오픈소스 포즈 추정 모델을 단순 적용하는 것이 아니라, PT 현장의 상담 데이터 구조, 촬영 품질 검증, 통증 반응 기록, 트레이너 검토 흐름, 재등록 상담 리포트 UX를 결합해 현장형 SaaS로 구현한다.

### 6.3 기술 검증 지표

| 검증 항목 | 지표 |
|---|---|
| 촬영 신뢰도 | PASS/CHECK/FAIL 비율, 재촬영률 |
| 반복 측정 안정성 | 동일 조건 반복 촬영 시 keypoint 오차 |
| 리포트 생성 성능 | 입력 완료 후 리포트 생성 성공률 |
| 트레이너 사용성 | 상담 준비 시간 감소 체감 |
| 설명 가능성 | 회원이 리포트를 이해했다고 응답한 비율 |
| 위험 신호 관리 | REVIEW/BLOCK 발생 시 트레이너 확인률 |
| 고객가치 | 재등록 상담 활용률, 유료 전환 의향 |

## 10. 브랜드 전략

### 10.1 장기 브랜드 정의

> LIGHT ONE은 회원의 몸 상태를 감으로 설명하지 않고, 운동 수행·통증 반응·자세 변화·생활습관 데이터를 기반으로 상담하고 관리하는 프리미엄 컨디셔닝 브랜드다.

### 10.2 권장 확장 순서

> 리포트 MVP → 기존 센터 파일럿 → 유료 PoC → LIGHT ONE 방법론 정리 → 교육·인증 → 제휴형 브랜드 → 자체 브랜드

### 10.3 피해야 할 순서

> 헬스장 오픈 → SaaS 개발 → 회원 모집 → 브랜드 확장

현재 조건에서는 오프라인 공간 선투자보다 리포트 MVP와 센터 파일럿이 우선이다.

### 11.3 방어 가능한 핵심 문장

> LIGHT ONE은 의료 진단이나 치료 목적의 서비스가 아니라, PT 현장에서 흩어져 있는 운동 수행·통증 반응·자세 변화·생활습관 데이터를 구조화하고, 트레이너가 검토할 수 있는 상담 리포트를 생성하여 회원 유지와 재등록 상담에 활용하는 비의료 웰니스 SaaS입니다.

> 기술적으로는 촬영 QC, 자세 keypoint, QS/JATC 지표, AUTO/REVIEW/BLOCK 라우팅을 활용하되, AI가 최종 판단하지 않고 트레이너가 검토하는 Human-in-the-Loop 구조로 설계합니다.

> 초기 사업성은 대규모 시장 수치가 아니라 센터 1~3곳 파일럿, 트레이너·원장 인터뷰, 샘플 리포트 사용률, 월 구독 전환 의향으로 검증하겠습니다.

## 14. 최종 투자자용 30초 피치

LIGHT ONE은 PT센터와 필라테스 센터가 회원의 운동 수행, 통증 반응, 자세 변화, 생활습관 데이터를 구조화해 트레이너 검토용 상담 리포트를 생성하는 비의료 웰니스 SaaS입니다. 기존 자세분석 앱이 “측정”에 집중했다면, LIGHT ONE은 실제 PT 현장의 상담, 변화 설명, 재등록, 회원 유지 문제를 해결합니다. 초기에는 센터 월 구독과 리포트 과금으로 검증하고, 이후 LIGHT ONE 컨디셔닝 방법론을 교육·인증 IP와 프리미엄 PT 브랜드로 확장합니다. 창업자는 PT 현장과 헬스장 운영 경험, 편집·리포트 제작 역량, 바이오헬스케어 AI 교육 기반의 개발 역량을 결합해 저비용 MVP와 현장 검증을 직접 수행할 수 있습니다.

## 15. 최종 판단

LIGHT ONE은 **“AI 체형·통증 분석”으로 밀면 위험하고, “PT 상담 리포트 SaaS + 컨디셔닝 브랜드 운영 시스템”으로 밀면 심사 방어력이 생긴다.**

최종 방향은 다음으로 고정한다.

> **비의료 PT 상담 리포트 SaaS**  
> **+ 촬영 QC 기반 신뢰도 보조 레이어**  
> **+ QS/JATC 상담 참고 지표**  
> **+ Human-in-the-Loop 트레이너 검토**  
> **+ 프리미엄 컨디셔닝 브랜드 확장**

사업성은 아직 확정이 아니다. 하지만 이 방향이면 “아이디어”가 아니라 **고객 문제, 기술 증거, 리스크 관리, 수익모델, 대표자 적합성**을 한 줄로 연결할 수 있다.

최종 권고는 **진행**이다. 단, 다음 30일 안에 반드시 고객검증과 샘플 리포트 피드백을 확보해야 한다. 고객검증이 없으면 좋은 문서로 끝나고, 고객검증이 있으면 지원사업·멘토링·투자 심사에서 방어 가능한 사업계획이 된다.
