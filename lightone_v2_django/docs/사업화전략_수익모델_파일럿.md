# LIGHT ONE V2 사업화 전략서 (수익모델·고객검증·파일럿 전략)

> **작성일:** 2026-07-01
> **작성자:** ONLY ONE 프로젝트 팀 (AI 전략·개발·기획 담당)
> **대상:** LIGHT ONE 리드 송광일 님
> **목적:** 모두의 창업 2기 지원 및 초기 사업화(MVP) 단계에서 심사위원과 잠재 고객을 설득할 수 있는 현실적이고 방어 가능한 비즈니스 모델 및 실행 계획 제시.

## 1. 비즈니스 핵심 가치 (Value Proposition)

LIGHT ONE은 **"데이터 기반 PT 의사결정지원 및 비의료 상담 리포트 SaaS"**입니다.
트레이너를 대체하는 것이 아니라, 트레이너의 감각적 판단을 **객관적 데이터(QS, RPE, NRS)**로 구조화하여 회원 관리 품질을 상향 평준화합니다.

### 1.1 해결하고자 하는 문제 (Pain Points)
- **회원(엔드 유저):** "PT를 30회나 받았는데 내 몸이 어떻게 좋아졌는지 객관적으로 알 수 없다." (높은 환불/이탈률)
- **트레이너(실무자):** "회원마다 체형과 통증이 다른데, 기록이 파편화되어 다음 세션 준비와 상담이 어렵다."
- **센터장(결제권자):** "트레이너의 개인 역량에 따라 PT 품질 편차가 너무 크다. 에이스 트레이너가 퇴사하면 회원이 대거 이탈한다."

### 1.2 LIGHT ONE의 해결책 (Solution)
- 트레이너의 세션 기록을 바탕으로 **QS(품질 점수) 산출 및 위험 라우팅(AUTO/REVIEW/BLOCK)** 제공.
- 회원에게 제공할 수 있는 **전문적인 비의료 상담 리포트** 자동 생성.
- 센터장에게는 소속 트레이너들의 전반적인 **회원 관리 지표 대시보드** 제공.

## 2. 수익 모델 전략 (Revenue Model)

초기에는 B2B(센터 단위) 구독 모델에 집중하여 안정적인 반복 수익(MRR)을 창출합니다. 향후 B2C(회원 개인) 프리미엄 리포트 판매로 확장합니다.

### 2.1 B2B SaaS 구독 모델 (Primary)
타겟 고객: 중소형 PT 샵 및 대형 피트니스 센터 원장/매니저

| 요금제 | 타겟 | 가격 (예상) | 핵심 기능 |
|--------|------|-------------|-----------|
| **Starter** | 1인 PT 샵, 프리랜서 트레이너 | ₩29,000 / 월 | 최대 30명 회원 관리, 기본 QS 산출, 기본 리포트 (월 50건) |
| **Pro** | 3~5인 규모 중소형 PT 샵 | ₩89,000 / 월 | 무제한 회원 관리, 트레이너별 권한 분리, 커스텀 로고 리포트 (무제한) |
| **Enterprise** | 대형 피트니스 센터, 프랜차이즈 | ₩199,000 / 월 | 지점 통합 대시보드, 트레이너 성과 지표(KPI) 연동, 전담 지원 |

*※ 초기 파일럿 기간(3~6개월)에는 Starter/Pro 기능을 한정된 센터에 무료 또는 파격 할인가로 제공하여 데이터를 확보합니다.*

### 2.2 확장 수익 모델 (Secondary)
- **B2B2C 프리미엄 리포트:** 기본 리포트 외에 심층 분석(변화 추세 상세 해설 등)이 포함된 프리미엄 리포트를 회원이 원할 경우, 센터를 통해 건당 과금(예: ₩5,000/건)하여 센터와 수익 분배.
- **데이터 API 제공:** 축적된 익명화 건강/운동 데이터를 피트니스 기기 제조사나 보험사(향후 규제 완화 시)에 API 형태로 제공.

## 3. 고객 검증 전략 (Customer Validation)

가설을 세우고 실제 현장에서 빠르게 검증(Build-Measure-Learn)하여 MVP를 개선합니다.

### 3.1 핵심 검증 가설
1. **가설 1 (트레이너):** 트레이너는 세션 직후 3분 이내에 데이터를 입력하는 수고를 감수할 것이다. (보상: 전문적인 상담 리포트 자동 생성)
2. **가설 2 (센터장):** 센터장은 트레이너의 PT 품질을 균일하게 유지하고 회원 재등록률을 높이기 위해 월 89,000원의 구독료를 지불할 의향이 있다.
3. **가설 3 (회원):** 회원은 LIGHT ONE 리포트를 받았을 때, 구두 상담만 받을 때보다 PT 서비스에 대한 만족도와 재등록 의향이 높아진다.

### 3.2 검증 방법론 (Lean Startup 방식)
- **1단계: 페이크 도어 (Fake Door) 테스트**
  - 랜딩 페이지 제작: "트레이너를 위한 데이터 기반 PT 상담 리포트 솔루션, LIGHT ONE"
  - 서비스가 완성되기 전, 사전 예약(Waitlist) 버튼을 만들어 트레이너/센터장의 클릭 전환율 및 이메일 수집률 측정.
- **2단계: 컨시어지 (Concierge) MVP**
  - 시스템이 완벽하지 않은 상태에서, 트레이너가 구글 폼/엑셀로 데이터를 입력하면 광일 님(팀)이 수동으로 리포트를 디자인하여 PDF로 전달.
  - 리포트를 받은 회원과 트레이너의 실제 반응 인터뷰.

## 4. 파일럿 테스트 실행 계획 (Pilot Strategy)

검증된 가설을 바탕으로 실제 환경에서 MVP를 테스트하고 레퍼런스를 확보합니다.

### 4.1 파일럿 대상 선정 (Target Audience)
- **규모:** 트레이너 3~5명 규모의 중소형 PT 샵 2~3곳.
- **조건:** 평소 기록 관리에 관심이 많고, 새로운 IT 도구 도입에 거부감이 적은 젊은 원장/매니저가 있는 곳. (광일 님의 기존 네트워크 적극 활용)

### 4.2 파일럿 진행 단계 (3개월 로드맵)

#### [1개월 차] 온보딩 및 데이터 수집 적응기
- **목표:** 트레이너들이 시스템에 데이터를 꾸준히 입력하는 습관 형성.
- **실행:**
  - 트레이너 대상 시스템 사용 교육 (입력은 3분 이내로 최소화).
  - 매주 데이터 입력률 모니터링 및 피드백 수렴.
  - UI/UX 불편 사항 즉각 개선.

#### [2개월 차] 리포트 활용 및 라우팅 검증기
- **목표:** 생성된 리포트가 실제 상담에 활용되고, 위험 라우팅(REVIEW/BLOCK)이 유효하게 작동하는지 확인.
- **실행:**
  - 월말/중간 평가 시 LIGHT ONE 리포트를 활용하여 회원 상담 진행.
  - 'REVIEW' 상태가 뜬 회원의 경우, 트레이너가 실제로 추가 주의를 기울이거나 프로그램(루틴)을 수정했는지 추적.
  - 의료적 표현(금지어)이 리포트에 노출되지 않는지 철저히 모니터링.

#### [3개월 차] 성과 측정 및 레퍼런스 확보기
- **목표:** 파일럿 도입 전/후의 정량적, 정성적 성과 도출.
- **실행:**
  - **정량 지표:** 회원 재등록률 변화, 환불률 변화, 트레이너 1인당 관리 회원 수 변화.
  - **정성 지표:** 회원 만족도 설문, 트레이너 업무 피로도 감소 여부 인터뷰.
  - 성공 사례(Use Case)를 발굴하여 마케팅 자료 및 투자 IR 자료로 가공.

## 5. 리스크 관리 (Risk Management)

| 리스크 유형 | 예상 문제 | 방어 전략 |
|-------------|-----------|-----------|
| **규제 리스크** | 비의료 건강관리서비스 범위를 벗어나 의료법 위반 소지 | 철저한 금지어 필터링, "본 리포트는 의학적 진단을 대체하지 않습니다" 명시, 보건복지부 유권 해석 사전 의뢰. |
| **운영 리스크** | 트레이너의 데이터 입력 귀찮음으로 인한 사용률 저하 | 모바일 친화적 UI, 음성 인식(STT) 입력 지원(향후), 필수 입력 항목 최소화. |
| **시장 리스크** | 유사한 PT 관리 앱(단순 예약/기록 앱)과의 차별성 부족 | 단순 기록이 아닌 **'데이터 기반 QS 산출 및 위험 라우팅'**이라는 의사결정지원 기능 강조. |
