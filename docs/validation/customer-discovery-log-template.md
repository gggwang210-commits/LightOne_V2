# 고객 발견 인터뷰 로그 템플릿

이 템플릿은 LIGHTONE V2의 고객 인터뷰와 현장 피드백을 일관되게 기록하기 위한 문서입니다. 질문지는 [트레이너·센터 고객 인터뷰 질문지](trainer-interview-script.md), 지불의사 검증은 [WTP 테스트 계획](wtp-test-plan.md), 파일럿 후보 관리는 [파일럿 센터 리스트 템플릿](pilot-center-list-template.md)을 참조합니다.

## 1. 기록 원칙

- 실제 개인정보를 기록하지 않습니다.
- 이름, 연락처, 사진, 상세 일정 등 식별 가능 정보는 별도 승인된 안전한 도구에만 보관합니다.
- 이 문서에는 `인터뷰-001`, `센터-A`, `트레이너-B` 같은 가상·비식별 표현만 사용합니다.
- 통증 관련 답변은 운동 중 반응 또는 상담 참고 메모로만 적습니다.
- 가격, 시장 규모, 성능 수치는 검증 전까지 `[확인필요]`로 표시합니다.
