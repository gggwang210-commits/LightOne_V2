# LIGHT ONE V2 초보자 실행 가이드 🚀

이 문서는 비전공자나 코딩 초보자도 LIGHT ONE V2 Django 프로젝트를 내 컴퓨터에서 쉽게 실행해 볼 수 있도록 작성되었습니다.

---

## 1. 프로그램 설치 (준비물)

Django를 실행하려면 **Python**과 코드를 편집할 **VS Code**가 필요합니다.

1. **Python 설치하기**
   - [Python 공식 홈페이지](https://www.python.org/downloads/) 접속
   - 다운로드 버튼 클릭 후 설치 파일 실행
   - ⚠️ **매우 중요:** 설치 화면 맨 아래에 있는 **`Add Python to PATH`** (또는 `Add Python 3.x to PATH`) 체크박스를 **반드시 체크**하고 Install Now 클릭!

2. **VS Code 설치하기**
   - [VS Code 공식 홈페이지](https://code.visualstudio.com/) 접속 및 설치

---

## 2. 프로젝트 열기

1. 다운로드 받은 `LIGHTONE_V2_Complete.zip` 파일의 압축을 풉니다.
2. VS Code를 실행합니다.
3. 상단 메뉴에서 `파일(File)` > `폴더 열기(Open Folder)`를 클릭합니다.
4. 압축을 푼 폴더(`lightone_v2_django`)를 선택하고 열기 버튼을 누릅니다.

---

## 3. 터미널 열고 명령어 입력하기

VS Code에서 상단 메뉴의 `터미널(Terminal)` > `새 터미널(New Terminal)`을 클릭합니다.
화면 아래쪽에 터미널 창이 열리면, 아래 명령어들을 **한 줄씩 복사해서 붙여넣고 엔터(Enter)**를 칩니다.

### 1단계: 가상환경 만들기
가상환경은 이 프로젝트만을 위한 독립된 공간을 만드는 작업입니다.
```bash
python -m venv .venv
```
*(명령어를 치고 5~10초 정도 기다리면 `.venv`라는 폴더가 생성됩니다.)*

### 2단계: 가상환경 켜기
- **Windows 사용자:**
  ```bash
  .venv\Scripts\activate
  ```
- **Mac/Linux 사용자:**
  ```bash
  source .venv/bin/activate
  ```
*(성공하면 터미널 입력창 왼쪽에 `(.venv)`라는 글자가 나타납니다.)*

### 3단계: 필요한 패키지 설치하기
```bash
pip install -r requirements.txt
```
*(화면에 글자들이 막 올라가면서 설치가 진행됩니다. 완료될 때까지 기다려주세요.)*

### 4단계: 데이터베이스 세팅 및 더미 데이터 넣기
이전에 만들어둔 가짜(더미) 회원과 트레이너, 세션 기록을 DB에 넣습니다.
```bash
python manage.py migrate
python setup_dummy.py
```

---

## 4. 서버 실행하고 접속하기! 🎉

마지막으로 아래 명령어를 입력하여 서버를 켭니다.
```bash
python manage.py runserver
```

터미널에 `Starting development server at http://127.0.0.1:8000/` 이라는 문구가 뜨면 성공입니다!

**웹 브라우저(크롬 등)를 열고 주소창에 아래 주소를 입력하세요:**
👉 `http://127.0.0.1:8000`

### 🔑 로그인 정보
- **아이디:** `admin`
- **비밀번호:** `admin`

## 💡 자주 묻는 질문 (트러블슈팅)

**Q. `python은(는) 내부 또는 외부 명령, 실행할 수 있는 프로그램... 이 아닙니다.` 라는 에러가 떠요.**
A. Python 설치 시 `Add Python to PATH`를 체크하지 않아서 발생합니다. Python을 지우고 다시 설치할 때 꼭 체크해 주세요.

**Q. 가상환경 켜기(`activate`)에서 보안 오류(PSSC 등)가 나요.**
A. Windows 보안 정책 때문입니다. 터미널 종류를 `PowerShell` 대신 `Command Prompt(cmd)`로 변경한 후 다시 시도해 보세요. (터미널 우측 상단 `+` 버튼 옆 화살표 클릭 -> Command Prompt 선택)

**Q. 서버를 끄고 싶어요.**
A. 터미널 창을 클릭한 상태에서 키보드의 `Ctrl + C`를 누르면 서버가 꺼집니다.
