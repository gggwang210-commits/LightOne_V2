# CHANGELOG — LIGHTONE Redesign V2

## 2026-07-02

### Added
- 신규 정적 랜딩 페이지 폴더 `LIGHTONE_redesigned_v2` 생성.
- 단일 HTML + hash route 구조 적용.
- Home, Problem, Solution, AI Analysis, Market, Business Model, Pilot Study, Roadmap, Public Submission, Promotion, Contact 페이지 추가.
- 공공기관·공모전 제출용 요약 페이지 추가.
- SNS 홍보용 Promotion 페이지 추가.
- 파일럿 연구 안내 페이지 추가.
- TAM/SAM/SOM 시장 규모 섹션을 모바일 세로 카드형, 데스크톱 3단 카드형으로 재디자인.
- 접근성 개선: skip link, aria-label, aria-current, 키보드 포커스 스타일, 44px 이상 터치 영역.
- SEO 및 SNS 공유 메타 추가.

### Changed
- 기존 LIGHT ONE 메시지를 “AI 체형분석 단독 상품”이 아니라 “비의료 PT 상담·컨디셔닝 관리 지원 솔루션” 중심으로 재정의.
- 의료적 효과를 단정하는 표현을 제거하고 “분석”, “관리 방향 제안”, “사용성 검토”, “컨디셔닝 케어 지원” 표현으로 정리.
- 시장 수치에는 출처 또는 `[확인필요]` 표시를 유지.
- 모바일 360px에서도 단어 쪼개짐과 가로 스크롤 위험을 줄이도록 레이아웃과 줄바꿈 정책 개선.

### 2026-07-02 Strategy Update
- 모두의 창업 2기 지원 전략서 기준으로 서비스 제목을 `PT 트레이너를 위한 AI 체형분석 상담 리포트 SaaS`로 정교화.
- 시장 섹션을 전국 피트니스 시설 수, 도입 가능 시설 수, 1년차 매출 목표 중심으로 재구성.
- 수익 모델을 B2B 구독, 리포트 단건 판매, 익명 집계 데이터 인사이트 3중 구조로 업데이트.
- 공공기관 제출 요약에 재등록 상담, 100점 리포트, AUTO/REVIEW/BLOCK 검토 우선순위, 의료기기 해당 여부 확인 필요 항목을 반영.
